# LeetCode Practice Workspace

This folder contains Python solutions for various LeetCode problems.

The Repo will be focused on solving question which will help to learn data struture and algorithms with a view of Embedded Software Engineer.

## Structure
- Each file is named after the problem, such as `single-number.py`.
- Solutions are written as standalone Python scripts with a `Solution` class.
- Some files include a short problem summary and key learnings in the header comment.

## Usage
- Open any `.py` file in this folder.
- Read the problem statement and approach in the header comment.
- Run the file directly to see the example output.

## Notes
- These are practice solutions and may not be the only or most optimal approach.
- Feel free to modify or extend them while learning.
